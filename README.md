# Rougelike Room Formatter

This tool allows the user to convert a 16 by 16 image into a list with numbers representing an object/character. There is currently support for 4 different objects.

## Future Features

The following features are planned to be added in the future:

- Save to file
- Export in CSV format
- Support for more objects
- Preview feature

### Possible Features

These features are not planned but may be added later:

- GUI
- Draw the image in the app
